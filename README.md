# HCI Project Setup
Setup/run instructions.

## Backend (Flask microtool for web dev with Python)
PowerShell:

- `cd backend`
- `python -m venv .venv`
- `.\.venv\Scripts\Activate.ps1`
- `pip install -r requirements.txt`
- `python app.py`

Backend runs at:

- `http://127.0.0.1:5000`

## Mobile (Expo React Native)
PowerShell:

- `cd mobile`
- `npm install`
- `npx expo start`
